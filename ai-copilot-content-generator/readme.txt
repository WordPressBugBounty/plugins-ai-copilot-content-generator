== Frequently Asked Questions ==

= What is AIWU? =
AIWU is an AI automation platform for WordPress that combines workflow automation, AI chatbots, content generation, AI forms, MCP integration and multi-model AI support in one plugin.

= Is AIWU an AI chatbot plugin for WordPress? =
Yes. AIWU includes a fully customizable AI Chatbot that can be trained on your content, recommend products, answer questions, capture leads and support visitors in real time.

= How does AIWU recommend products and posts in the chatbot? =
AIWU uses function calling and real-time context from your WordPress and WooCommerce data. The AI Chatbot can search products, posts, categories and metadata, call internal functions, and return relevant recommendations as interactive cards directly in the conversation. This allows the chatbot to act as a true sales and content assistant, not just a text responder.

= Can I build automation workflows like in n8n or Zapier? =
Yes. AIWU includes a visual drag & drop Workflow Builder (n8n-style) that runs natively inside WordPress and lets you automate content, WooCommerce, marketing, support and custom processes. [Learn more about Workflow Builder](https://aiwuplugin.com/ai-workflow-automation-builder-for-wordpress/)

= Which AI models does AIWU support? =
AIWU works with OpenAI (ChatGPT), Claude, Google Gemini, Perplexity, DeepSeek and OpenRouter. You can use your own API keys and choose different models for different tasks and workflows.

= Can I use my own data to train the AI? =
Yes. AIWU supports fine-tuning and embeddings (RAG), so you can train AI models on your own website content, documents and datasets.

= What is MCP and how does it work with WordPress? =
MCP (Model Context Protocol) allows AI assistants like ChatGPT or Claude to directly control your WordPress site via natural language: create pages, edit posts, manage WooCommerce products, users and settings. [Learn more about MCP Integration](https://aiwuplugin.com/claude-mcp-integration/)

= Can AIWU generate content and automate SEO? =
Yes. AIWU can generate posts in bulk, run RSS autoblogging, optimize content, add internal links, update existing posts and automate WooCommerce product content.

= Does AIWU work with WooCommerce? =
Yes. You can generate and update product titles, descriptions, images, categories, tags, reviews and automate WooCommerce workflows with AI and visual automation.

= Can I create AI-powered forms and quizzes? =
Yes. AIWU includes an AI Forms Builder to create smart forms, surveys, quizzes and calculators with conditional logic and AI-generated results. [Learn more about AI Forms](https://aiwuplugin.com/ai-forms-builder/)

= Is AIWU a SaaS or does it run inside WordPress? =
AIWU runs entirely inside your WordPress site. There is no external SaaS, no middle platform and no usage credits from us. You only use your own AI provider API keys.

= Is AIWU suitable for developers? =
Yes. AIWU provides REST API, JavaScript API, PHP helpers and MCP support to build custom AI agents, automations and integrations. [Explore Developer API](https://aiwuplugin.com/knowledge-base/)

= Can I build AI agents with AIWU? =
Yes. Using the Workflow Builder, MCP integration and Developer API, you can create AI agents that interact with your site, tools and data.

= Does AIWU replace multiple plugins? =
Yes. AIWU combines AI chatbot, workflow automation, content generation, SEO automation, AI forms, WooCommerce AI and MCP into one unified platform.

= Do I need coding skills to use AIWU? =
No. Most features are no-code and work through visual builders and templates. Developers can extend everything using APIs if needed.
